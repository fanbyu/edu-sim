# Edu-Sim：学校垂直领域教育多智能体预测系统愿景文档

## 1. 核心定位

**Edu-Sim** 不仅仅是一个仿真工具，而是一个**面向学校垂直领域的群智预测引擎**。

它继承了 MiroFish 的核心技术优势（多智能体协作、知识图谱、行为演化），但将应用场景从“社会舆情预测”精准聚焦于“教育生态推演”。其核心目标是利用 AI 智能体模拟学校生态中的各类角色（学生、教师、家长、管理者），通过构建高保真的教育知识图谱和并行仿真教学互动，**提前预测**教育改革、教学策略或校园事件在特定群体中的演化趋势与最终效果。

---

## 2. 为什么需要“群智预测”？

在教育场景中，单一维度的线性预测往往失效，因为教育本质上是一个复杂的**社会-心理系统**。Edu-Sim 的“群智”价值体现在以下三个维度：

*   **🔮 趋势预测 (Trend Forecasting)：** 能够提前预判新教学法、考试改革或校园政策在学生群体中的接受度、执行阻力及最终成效。
*   **🧠 群智涌现 (Swarm Emergence)：** 捕捉个体行为汇聚成的集体模式。例如，某种干预措施可能在少数学生身上有效，但在群体中却引发了普遍的焦虑或抵触情绪。
*   **🎓 精准干预 (Precision Intervention)：** 基于 IRT（项目反应理论）和心理测量模型，为每个学生生成最优的成长路径，并为老师提供数据驱动的决策支持。

---

## 3. 核心功能：作业前置评估与影响预测

近期最核心的业务目标是实现**“老师在布置作业前即可预测题目对学生的影响”**。系统需要具备以下能力：

### 3.1 三维预测坐标系
为了实现高精度的事前预测，系统必须综合掌握以下三类信息：
1.  **学情维度 (Student State)：** 掌握学生过去做过的试题、得分情况、错题分布以及动态更新的 IRT 能力值 ($\theta$)。
2.  **进度维度 (Curriculum Context)：** 感知老师当前的讲课进度，识别哪些是刚讲的知识点（短期记忆强），哪些是旧知识（受遗忘曲线影响）。
3.  **试题维度 (Item Characteristics)：** 解析准备预测的新试题，提取其难度、区分度、认知层级及考察的知识点标签。

### 3.2 预测输出指标
系统应能针对一套试题给出明确的定性结论和定量分析：
*   **整体有效性：** 预测该作业对全班/全校掌握度的提升贡献（正向明显 / 正向不明显 / 无效 / 负向）。
*   **分层影响分析：** 预测优等生、中等生和后进生分别会受到怎样的影响。
*   **风险预警：** 预测是否会引发班级平均焦虑指数飙升或产生大面积挫败感。
*   **优化建议：** 根据预测结果，自动推荐修改题目的方向（如：增加引导提示、降低计算量等）。

## 4. 核心技术架构支撑

为了实现上述愿景，Edu-Sim 构建了以下核心技术模块：

| 模块名称 | 核心职责 | 关键技术 |
| :--- | :--- | :--- |
| **高保真 Agent 建模** | 模拟多元主体博弈 | 学生 Agent (IRT + 心理画像)、教师 Agent (教学风格 + 效能感) |
| **动态教育知识图谱** | 追踪演化路径 | Graphify 持久化存储、知识点逻辑关系、教学时间轴 |
| **平行仿真推演引擎** | 寻找未来分支 | 基于 OASIS 的多智能体并行交互、蒙特卡洛模拟 |
| **闭环反馈机制** | 自动优化策略 | 预测结果 -> 智能推荐 -> 效果再评估 |
| **通用 LLM 驱动层** | 深度语义理解 | 支持 Qwen、豆包等国内主流大模型进行试题解析与报告生成 |

## 5. 典型应用场景示例

| 场景 | 输入 (现实种子) | 预测目标 (演化结果) |
| :--- | :--- | :--- |
| **新题布置前评估** | 某套期末复习题 + 当前班级学情快照 | 预测 30 天后不同能力层学生的成绩波动及心理压力变化。 |
| **教学策略选择** | 两种不同的课堂导入方案 | 预测哪种方案更能激发全班的学习动机并降低畏难情绪。 |
| **个性化路径推荐** | 学生 S001 的知识盲区 + 学习风格 | 预测采用“支架式教学”后，该生在下个单元的掌握度提升概率。 |

## 6. 总结与展望

Edu-Sim 致力于成为学校管理者和一线教师的**“数字沙盘”**。通过将复杂的教育过程数字化、模型化，我们希望在真实的课堂发生之前，就能在虚拟空间中完成成百上千次的推演，从而让每一次教学决策都更加科学、精准且充满人文关怀。

> **下一步计划：** 重点完善 `_apply_learning_context` 逻辑，实现基于知识图谱的教学进度匹配与遗忘衰减算法，让预测结果真正贴合学生的实时学情。
